# BRUMA - Branding Assets

Coloca aquí los archivos de logo de BRUMA:

## Archivos recomendados:
- `logo.svg` - Logo principal en SVG (vectorial)
- `logo.png` - Logo en PNG (alta resolución)
- `logo-white.svg` - Logo en blanco para fondos oscuros
- `favicon.ico` - Favicon para el navegador

## Colores de marca:
- **Main (Logo, acentos):** `#004b49` - Verde azulado oscuro
- **Secondary (Background):** `#f5f2e9` - Crema cálido

## Uso:
Los logos se importarán automáticamente desde esta carpeta en:
- Navbar
- Sidebar del dashboard
- Página de login
- Favicon del sitio
